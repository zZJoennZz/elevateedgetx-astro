# Elevate Edge TX

A modern web project built with Astro.

## Getting Started

### Prerequisites
- Node.js 16.x or higher
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure

```
src/
    pages/       # Route pages
    components/  # Reusable components
    layouts/     # Layout templates
public/        # Static assets
```

## Technologies

- **Astro** - Static site generator
- **Node.js** - JavaScript runtime
- **TailwindCSS** - Styling
